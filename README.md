#  Jogo da Velha em Python

Projeto de **Jogo da Velha (Tic-Tac-Toe)** desenvolvido em **Python**, executado no terminal.  
Criado com o objetivo de praticar **lógica de programação**, estruturas de repetição, condicionais, listas e funções.

---

##  Sobre o projeto

Este jogo permite que **dois jogadores** joguem alternadamente no terminal, utilizando os símbolos **X** e **O**.  
O programa valida as jogadas, verifica condições de vitória e empate, e exibe o tabuleiro a cada rodada.

Projeto desenvolvido para **fins educacionais**, ideal para iniciantes em Python.

---

##  Conceitos praticados

- Lógica de programação  
- Estruturas condicionais (`if`, `elif`, `else`)  
- Laços de repetição (`while`)  
- Listas  
- Funções  
- Validação de entrada do usuário  

---

##  Tecnologias utilizadas

- Python 3
